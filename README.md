# java-web-maven-spring-jsp-db2-simple

## Description
A POC for springboot web app with jsp support
connecting to a database.

## Tech stack
- java
- maven
  - hibernate
  - springboot
  - jsp
  - db2 connector
- bootstrap
- jquery
- dataTable

## Docker stack
- ibmcom/db2
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
